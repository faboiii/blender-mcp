# Recommended GitHub Repositories

A curated list of GitHub repositories relevant to BlenderMCP and Model Context Protocol integrations.

## Model Context Protocol (MCP) Servers & Tools

### Official & Core MCP Resources
- **[modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers)** - Official collection of reference MCP server implementations
- **[modelcontextprotocol/python-sdk](https://github.com/modelcontextprotocol/python-sdk)** - Official Python SDK for building MCP servers
- **[modelcontextprotocol/typescript-sdk](https://github.com/modelcontextprotocol/typescript-sdk)** - Official TypeScript SDK for MCP

### MCP Server Implementations
- **[modelcontextprotocol/create-python-server](https://github.com/modelcontextprotocol/create-python-server)** - Template for creating Python MCP servers
- **[wong2/mcp-server-fetch](https://github.com/wong2/mcp-server-fetch)** - MCP server for web content fetching
- **[executeautomation/mcp-servers](https://github.com/executeautomation/mcp-servers)** - Collection of various MCP server implementations
- **[MarkusMeyer1/arxiv-mcp-server](https://github.com/MarkusMeyer1/arxiv-mcp-server)** - ArXiv integration via MCP
- **[calclavia/mcp-obsidian](https://github.com/calclavia/mcp-obsidian)** - Obsidian knowledge base integration

### MCP Development Tools
- **[ktanaka101/mcp-server-deno](https://github.com/ktanaka101/mcp-server-deno)** - Deno-based MCP server implementation
- **[blazickjp/mcp-server-kubernetes](https://github.com/blazickjp/mcp-server-kubernetes)** - Kubernetes cluster management via MCP
- **[QuantGeekDev/mcp-server-docker](https://github.com/QuantGeekDev/mcp-server-docker)** - Docker container management via MCP

## Blender Automation & Integration

### Blender Python APIs & Addons
- **[blender/blender](https://github.com/blender/blender)** - Official Blender source code (essential reference for API understanding)
- **[carson-katri/dream-textures](https://github.com/carson-katri/dream-textures)** - AI-powered texture generation in Blender using Stable Diffusion
- **[3IXAMSOFTWARE/3IXAM](https://github.com/3IXAMSOFTWARE/3IXAM)** - Blender fork with additional features
- **[Moo-Ack-Productions/MCprep](https://github.com/Moo-Ack-Productions/MCprep)** - Minecraft world imports (great addon architecture reference)

### Blender Scripting & Automation
- **[njanakiev/blender-scripting](https://github.com/njanakiev/blender-scripting)** - Collection of Blender scripting examples
- **[CGArtPython/bpy_building_blocks](https://github.com/CGArtPython/bpy_building_blocks)** - Reusable Python code blocks for Blender
- **[domlysz/BlenderGIS](https://github.com/domlysz/BlenderGIS)** - Geographic data in Blender (shows complex addon patterns)

### Blender-AI Integration
- **[Zev-G/blender-diffusion](https://github.com/Zev-G/blender-diffusion)** - Stable Diffusion integration for Blender
- **[charlierix/BlenderVision](https://github.com/charlierix/BlenderVision)** - Computer vision tools for Blender

## 3D Model Generation & AI

### AI 3D Model Generation
- **[threestudio-project/threestudio](https://github.com/threestudio-project/threestudio)** - Unified framework for 3D content generation
- **[ashawkey/stable-dreamfusion](https://github.com/ashawkey/stable-dreamfusion)** - Text-to-3D synthesis
- **[THUDM/CogVideo](https://github.com/THUDM/CogVideo)** - Text-to-video generation (relevant for animation workflows)

### 3D Asset Management
- **[sketchfab/sketchfab-python-wrapper](https://github.com/sketchfab/sketchfab-python-wrapper)** - Python wrapper for Sketchfab API (already integrated!)
- **[polyhaven/Public-API](https://github.com/polyhaven/Public-API)** - Poly Haven API documentation (already integrated!)

## Socket Programming & Networking

### Python Socket Servers
- **[encode/uvicorn](https://github.com/encode/uvicorn)** - Lightning-fast ASGI server implementation
- **[pallets/flask](https://github.com/pallets/flask)** - Lightweight web framework (useful for REST API alternatives)
- **[python-websockets/websockets](https://github.com/python-websockets/websockets)** - WebSocket implementation for Python

### Real-time Communication
- **[socketio/socket.io](https://github.com/socketio/socket.io)** - Real-time bidirectional event-based communication
- **[miguelgrinberg/python-socketio](https://github.com/miguelgrinberg/python-socketio)** - Python Socket.IO server and client
